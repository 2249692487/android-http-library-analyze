# android-http-library-analyze
Analysis and comparison about the mainstream Http library.

通过程序中对同一个网站使用不同的网络框架进行同步及异步的访问，使用Power Point制作了名为**Android-http-library-Analyze.pptx**的文档。想了解更多的比对结果可以打开项目中此文档阅读。
